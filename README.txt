Для работы программы необходимо наличие двух файлов: groups.txt и educators.txt, которые содержат информацию о номерах групп и именах преподавателей.

Файл groups.txt состоит из набора строк, каждая из которых содержит идентификатор группы (без года поступления!) и ее номер (без номера курса!), разделенные пробелом, например:

С01-мм 11
Б08-мм 42

Файл educators.txt состоит из набора строк, каждая из которых содержит имя или аббревиатуру преподавателя и его идентификатор, разделенные пробелом, например:

ИЖ 2657

— эта строка означает, что расписание Игоря Жукова доступно по адресу https://timetable.spbu.ru/EducatorEvents/2657
